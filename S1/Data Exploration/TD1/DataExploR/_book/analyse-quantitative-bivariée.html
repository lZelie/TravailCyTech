Analyse exploratoire des données

3 Analyse quantitative bivariée

Dans ce chapitre, on considère :

  • un corpus de données de \(N\) individus ;
  • deux variables quantitatives \(X\) et \(Y\) sur ce corpus de données.

Pour tout entier \(i\) entre \(1\) et \(N\), on note respectivement \(x_i\) et \(y_i\) la valeur des variables \(X\) et \(Y\) pour le \(i\)-ème individu.

3.1 Covariance. Coefficient de corrélation

3.1.1 Covariance

Définition 3.1 La covariance de \(X\) et \(Y\), notée \(c_{XY}\), est définie par \[c_{XY} = \dfrac{1}{N} \displaystyle\sum_{i = 1}^N \left(x_i - \overline{X}\right) \cdot \left(y_i - \overline{Y}\right)\]
Remarque. Le terme de “covariance” (variance croisée) se justifie par le fait que \(c_{XX} = \sigma_X^2\).

En R : Calculer la covariance de deux variables quantitatives

On utilise la fonction cov.
Mais pour des raisons qui seront vues en seconde année, cov calcule l’expression \[\dfrac{1}{N - 1}\displaystyle\sum_{i = 1}^N \left(x_i - \overline{X}\right) \cdot \left(y_i - \overline{Y}\right) = \dfrac{N}{N - 1} c_{XY}\] On construit alors une fonction cov.p qu calcule \(c_{XY}\) :

cov.p <- function(x, y = NULL, use = "everything")
 {
   cov(x, y, use = use) * (length(x) - 1) / length(x)
 }

(La fonction length calcule la longueur d’un vecteur.)
L’option use spécifie comment traiter les valeurs manquantes (NA) :

  • use = "everything" garde les valeurs manquantes et renvoie NA si il y a au moins une valeur manquante ;
  • use = "all.obs" garde les valeurs manquantes et renvoie une erreur si il y a au moins une valeur manquante ;
  • use = "na.or.complete" ne prend pas en compte les individus dont la valeur de l’une des deux variables est manquante.
cov.p(data$Continue, data$Discrète, use = "everything")
[1] NA
cov.p(data$Continue, data$Discrète, use = "all.obs")
Error in cov(x, y, use = use): observations manquantes dans cov / cor
cov.p(data$Continue, data$Discrète, use = "na.or.complete")
[1] 4.082453
Proposition 3.1 Soit \(e\) le vecteur de longueur \(N\) dont toutes les composantes sont égales à un : \(e = (1, 1, \ldots, 1)\). Alors \(c_{XY} = \dfrac{1}{N}(X - \bar{X} \cdot e)^\top (Y - \bar{Y} \cdot e)\).

3.1.2 Coefficient de corrélation linéaire

Définition 3.2 Le coefficient de corrélation linéaire de \(X\) et \(Y\), notée \(r_{XY}\), est définie par \[r_{XY} = \dfrac{c_{XY}}{\sigma_X \cdot \sigma_Y}\]


Le terme “coefficient” est justifié par le résultat suivant :

Proposition 3.2 On a toujours \(r_{XY} \in [-1, 1]\).

IMPORTANT. Le coefficient de corrélation linéaire s’interprète comme le cosinus de l’angle formé par les variables centrées réduites de \(X\) et \(Y\) dans l’espace des variables.

Le coefficient de corrélation linéaire permet donc de décrire la dépendance linéaire entre les variables \(X\) et \(Y\) :

  • \(r_{XY} = 0\) caractérise l’absence de corrélation affine : cela ne fournit aucune information quant à une absence ou une présence de corrélation non-affine ;
  • \(r_{XY} = \pm1\) caractérise une corrélation affine totale.

Lorsque \(r_{XY}\) n’est pas trop faible, son signe permet de décrire les monotonies relatives de \(X\) et \(Y\) :

  • \(r_{XY} > 0\) caractérise une évolution de \(X\) et \(Y\) dans le même sens ;
  • \(r_{XY} < 0\) caractérise des évolutions de \(X\) et \(Y\) opposées.

En R : Calculer le coefficient de corrélation linéaire de deux variables quantitatives

On utilise la fonction cor :

  • use spécifie comment traiter les valeurs manquantes (NA) :
    • use = "everything" garde les valeurs manquantes et renvoie NA si il y a au moins une valeur manquante ;
    • use = "all.obs" garde les valeurs manquantes et renvoie une erreur si il y a au moins une valeur manquante ;
    • use = "na.or.complete" ne prend pas en compte les individus dont la valeur de l’une des deux variables est manquante.
cor(data$Continue, data$Discrète, use = "everything")
[1] NA
cor(data$Continue, data$Discrète, use = "all.obs")
Error in cor(data$Continue, data$Discrète, use = "all.obs"): observations manquantes dans cov / cor
cor(data$Continue, data$Discrète, use = "na.or.complete")
[1] 0.1480895
Proposition 3.3 Soit \(X^*\) et \(Y^*\) les variables centrées réduites de \(X\) et \(Y\) respectivement. Alors \(r_{XY} = \dfrac{1}{N}(X^*)^\top Y^*\).
Remarque. Donc \(r_{XY} = c_{X^*,Y^*}\).

3.2 Régression linéaire

3.2.1 Calcul du modèle

Lorsque \(|r_{XY}|\) est proche de 1, on souhaite expliciter la relation affine \(Y \simeq aX + b\) qui décrit le mieux la corrélation entre les deux variables \(X\) et \(Y\)

Définition 3.3 Pour une droite \(y = ax + b\), on définit l’erreur quadratique moyenne de régression comme la moyenne des carrés des écarts entre les valeurs observées (i.e. \(Y\)) et les valeurs prévues (i.e. \(aX + b\)), c’est-à-dire \(\dfrac{1}{N} \displaystyle\sum_{i = 1}^N [y_i - (a x_i + b)]^2\).

Définition 3.4 On définit le modèle de régression linéaire de \(Y\) en \(X\) comme étant la droite qui minimise l’erreur quadratique moyenne de régression.
Théorème 3.1 Le modèle de regression linéaire \(y = \hat{a} x + \hat{b}\) a pour coefficients :

\(\hat{a} = \dfrac{c_{XY}}{\sigma_X^2}\qquad\) et \(\qquad \hat{b} = \overline{y} - \hat{a} \cdot \overline{x}\)

En R : Calculer le modèle de régression linéaire entre deux variables quantitatives.

On utilise la fonction lm (pour Linear Model) :

  • data spécifie le tableau de données éventuel utilisé comme référence : ainsi, dans notre exemple :

    lm(Continue ~ Discrète, data = data, ...)

    est équivalent à :

    lm(data$Continue ~ data$Discrète, ...)

  • na.action spécifie comment traiter les valeurs manquantes (NA) :

    • na.action = na.fail garde les valeurs manquantes et renvoie une erreur si il y a au moins une valeur manquante ;
    • na.action = na.omit ne prend pas en compte les individus dont la valeur de l’une des deux variables est manquante.
lm(Continue ~ Discrète, data = data, na.action = na.fail)
Error in na.fail.default(structure(list(Continue = c(-1.59, -4.12, 1.81, : missing values in object
lm(Continue ~ Discrète, data = data, na.action = na.omit)

Call:
lm(formula = Continue ~ Discrète, data = data, na.action = na.omit)

Coefficients:
(Intercept)     Discrète  
    -1.8787       0.1587

Les coefficients du modèle sont disponibles dans le champ coefficients qui est un vecteur de longueur deux contenant, dans l’ordre, l’ordonnée à l’origine et la pente du modèle :

linreg <- lm(Continue ~ Discrète, data = data, na.action = na.omit)
linreg$coefficients
(Intercept)    Discrète 
 -1.8786672   0.1586622

Donc le modèle de régression linéaire de Continue en Discrète est \[y = 0.1586622 \cdot x -1.8786672\]

3.2.2 Analyse du modèle

Définition 3.5 On définit les deux variables quantitatives suivantes :

  • \(\hat{Y} = \hat{a}X + \hat{b}\) est la variable des prévisions, c’est-à-dire des valeurs prévues par le modèle de régression linéaire : on note \(\hat{y}_i\) la valeur de \(\hat{Y}\) pour l’individu n°\(i\) ;
  • \(E = Y - \hat{Y}\) est la variable des résidus, c’est-à-dire des écarts entre les valeurs observées et les valeurs prévues : on note \(e_i\) la valeur de \(E\) pour l’individu n°\(i\).

Proposition 3.4

  • \(\overline{\hat{Y}} = \overline{Y}\)
  • \(\sigma_{\hat{Y}}^2 = r_{XY}^2 \cdot \sigma_Y^2\)
Définition 3.6 Le coefficient de détermination du modèle de régression linéaire \(y = \hat{a}x + \hat{b}\) est défini par le carré du coefficient de corrélation linéaire, c’est-à-dire \(r_{XY}^2\).


Le coefficient de détermination est donc la proportion de la variance totale expliquée par le modèle de régression linéaire. Autrement dit, le modèle de régression linéaire explique d’autant mieux les données que le coefficient de détermination est grand (i.e. que le coefficient de corrélation linéaire est grand).

Corollaire 3.1

  • \(\overline{E} = 0\)
  • \(\sigma_E^2 = (1 - r_{XY}^2) \cdot \sigma_Y^2\)
Théorème 3.2 (Formule de la décomposition de la variance) \(\sigma_Y^2 = \sigma_{\hat{Y}}^2 + \sigma_E^2\)

Autrement dit, la variance totale \(\sigma_Y^2\) se décompose en deux termes :

  • la variance \(\sigma_{\hat{Y}}^2\) expliquée par le modèle de régression linéaire ;
  • la variance résiduelle \(\sigma_E^2\) (donc non expliquée).

3.2.3 Validation du modèle

Visualisation des données et du modèle

On peut commencer par une validation visuelle avec le tracé des points et du modèle de régression linéaire calculé.

En R : Visualiser le nuage de points et le modèle de régression linéaire de deux variables quantitatives

On utilise la fonction plot pour afficher les points :

  • le premier paramètre est le vecteur des abscisses des points à tracer ;
  • le deuxième paramètre est le vecteur des ordonnées des points à tracer, de même longueur que le premier paramètre ;
  • main définit le titre du graphique ;
  • xlab définit la légende en abscisse ;
  • ylab définit la légende en ordonnée.

On utilise également la fonction abline pour ajouter une droite :

  • on peut spécifier les coefficients de la droite de trois manières différentes :

    • on spécifie les coefficients séparément :

      • l’ordonnée à l’origine d’abord, puis la pente dans le cas d’une droite usuelle ;
      • h spécifie la valeur dans le cas d’une droite horizontale ;
      • v spécifie la valeur dans le cas d’une droite verticale ;

    • coef spécifie le vecteur des deux coefficients : ordonnée à l’origine puis pente ;

    • reg spécifie un objet donc le champ coef ou coefficients contient les deux coefficients. Autrement dit :

      abline(reg = linreg)

      est équivalent à :

      abline(coef = linreg$coefficients)

      qui est également équivalent à :

      abline(linreg$coefficients[1], linreg$coefficients[2])
  • col spécifie la couleur utilisée pour le tracé de la droite ;
  • lwd spécifie l’épaisseur de la droite tracée (la valeur par défaut est 1).
plot(data$Discrète, data$Continue,
     main = "Continue en fonction de Discrète",
     xlab = "Discrète", ylab = "Continue")
abline(reg = linreg, col = "red", lwd = 2)

Visualisation des résidus

Un modèle de régression linéaire explique bien les données si il ne reste plus d’information dans les résidus. On va alors vérifier visuellement les points suivants :

  1. la moyenne des résidus est nulle (ce qui est le cas en théorie, cf. corollaire 3.1) ;
  2. la variance des résidus est constante ;
  3. les résidus ne sont pas corrélés.

En R : Visualiser les résidus du modèle de régression linéaire de deux variables quantitatives

On récupère le champ residuals du résultat issu de l’appel de lm :

mean(linreg$residuals)
[1] 1.605858e-16
plot(linreg$residuals, main = "Résidus",
     xlab = "Individu", ylab = "Résidu")

Calcul des leviers

Un modèle sert évidemment à décrire le lien entre les valeurs observées des variables \(X\) et \(Y\). Mais on souhaite également valider la qualité du modèle lors de l’insertion d’un nouvel individu.

Pour cela, il faut détecter les individus influents.
Définition 3.7 Un individu est dit influent par rapport au modèle de régression si et seulement si une légère modification de ses valeurs provoque une modification importante des paramètres dudit modèle.


On évalue l’influence de chaque individu de la manière suivante :

  1. on le retire du corpus des données ;
  2. on calcule le nouveau modèle de régression ;
  3. on calcule le résidu de l’individu par rapport à ce nouveau modèle.

Un résidu important caractérise un individu influent.

Proposition 3.5 Soit \(i\) un entier naturel entre \(1\) et \(N\).

Alors le résidu de l’individu n°\(i\) par rapport au modèle de régression linéaire obtenu sans cet individu est : \(\dfrac{e_i}{1 - \eta_i}\) avec \(\eta_i = \dfrac{1}{N} + \dfrac{1}{N-1} \cdot \dfrac{\left(x_i - \overline{X}\right)^2}{\sigma_x^2}\).

\(\eta_i\) est dit le levier de l’individu n°\(i\) dans le modèle de régression.

Remarque. On a \(\eta_i \in \left[\dfrac{1}{N},1\right]\). Étudions les cas extrêmes :

  • \(\eta_i = \dfrac{1}{N} \iff x_i = \overline{X}\) : dans un tel cas, l’individu n’est pas influent.
  • \(\eta_i = 1 \iff x_i \neq \overline{X}\) et \(x_j = \overline{X}\) pour tout \(j \neq i\) : dans un tel cas, l’individu est le seul individu influent.
Donc une valeur élevée de \(\eta_i\) traduit un individu influent.

En R : Calculer les leviers des invididus dans un modèle de régression linéaire

La fonction influence effectue plusieurs mesures et renvoie les résultats dans un objet. Le champ hat contient le vecteur des leviers :

Définition 3.8 Soit \(i\) un entier naturel entre \(1\) et \(N\).

On définit le résidu standardisé de l’individu \(i\) dans le modèle de régression linéaire \(\dfrac{e_i}{\hat{\sigma_E}\cdot \sqrt{1 - \eta_i}}\) où \(\hat{\sigma_E}\) est une estimation de la variance

Théorème 3.3 Sous l’hypothèse de normalité des résidus, le résidu standardisé d’un individu peu influent est compris entre \(-2\) et \(+2\).

3.3 Régressions non-linéaires

Pour identifier une régression non-linéaire \(Y \simeq f(X)\), on tente de se ramener à une régression linéaire entre les variables \(X'\) et \(Y'\) obtenues par transformation des variables \(X\) et \(Y\).

Exemples de traitement de régressions non-linéaires
Régression Transformation \(X\) Transformation \(Y\) Régression linéaire
\(Y \simeq b \cdot e^{aX}\) \(X' = X\) \(Y' = \ln Y\) \(Y' = aX' + \ln b\)
\(Y \simeq b \cdot X^a\) \(X' = \ln X\) \(Y' = \ln Y\) \(Y' = aX' + \ln b\)
\(Y \simeq \dfrac{a}{X} + b\) \(X' = \dfrac{1}{X}\) \(Y' = Y\) \(Y' = aX' + b\)
\(Y \simeq \dfrac{1}{1 + e^{-(aX + b)}}\) \(X' = X\) \(Y' = \ln\dfrac{Y}{1 - Y}\) \(Y' = aX' + b\)
