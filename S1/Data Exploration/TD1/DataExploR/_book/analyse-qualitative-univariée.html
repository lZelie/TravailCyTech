6 Analyse qualitative univariée

Dans ce chapitre, on considère :

  • un corpus de données de \(N\) individus ;
  • une variable qualitative \(X\) sur ce corpus de données.

Pour tout entier \(i\) entre \(1\) et \(N\), on note \(x_i\) la valeur de la variable \(X\) pour le \(i\)-ème individu.

On reprend l’exemple donné au début du document ;

6.1 Terminologie

Définition 6.1 Une modalité de \(X\) est une valeur de \(X\) prise par au moins un individu.

En R : Calculer l’ensemble des modalités d’une variable qualitative

On utilise la fonction levels :

levels(data$Ordinale)
[1] "1" "2" "3" "4"
levels(data$Nominale)
[1] "Bleu"  "Rouge" "Vert" 
levels(data$Dichotomique)
[1] "Non" "Oui"

6.1.1 Effectifs. Fréquences

Définition 6.2 Soit \(x\) une modalité de \(X\).

L’effectif de \(x\) est le nombre d’individus dont la valeur de \(X\) est \(x\), c’est-à-dire \[\operatorname{card}\{i \in \{1, \ldots, N\},\: x_i = x\}\]

En R : Calculer les effectifs d’une variable qualitative

On peut utiliser la fonction table :

table(data$Ordinale)

1 2 3 4 
5 8 9 8 
table(data$Nominale)

 Bleu Rouge  Vert 
   11    13     6

On peut alors récupérer l’effectif d’une modalité à partir du résultat :

table(data$Nominale)["Bleu"]
Bleu 
  11

L’option useNA spécifie comment traiter les valeurs manquantes NA :

  • no ne tient pas compte des valeurs manquantes dans le calcul ;
  • ifany tient compte des valeurs manquantes dans le calcul, s’il y en a ;
  • always tient compte des valeurs manquantes dans le calcul même s’il n’y en a pas.
table(data$Dichotomique, useNA = "no")

Non Oui 
 13  15 
table(data$Dichotomique, useNA = "ifany")

 Non  Oui <NA> 
  13   15    2 
table(data$Nominale, useNA = "always")

 Bleu Rouge  Vert  <NA> 
   11    13     6     0

Définition 6.3 Soit \(x\) une modalité de \(X\).

La fréquence de \(x\) est la proportion d’individus dont la valeur de \(X\) est \(x\), c’est-à-dire \[\dfrac{\operatorname{card}\{i \in \{1, \ldots, N\},\: x_i = x\}}{N}\]

En R : Calculer les fréquences d’une variable qualitative

On peut appliquer la fonction prop.table au résultat de table :

prop.table(table(data$Ordinale))

        1         2         3         4 
0.1666667 0.2666667 0.3000000 0.2666667 
prop.table(table(data$Nominale))

     Bleu     Rouge      Vert 
0.3666667 0.4333333 0.2000000

On peut alors récupérer la fréquence d’une modalité à partir du résultat :

prop.table(table(data$Nominale))["Bleu"]
     Bleu 
0.3666667

6.1.2 Représentations graphiques

6.1.2.1 Diagramme en bâtons

Pour chaque modalité, on trace un bâton dont la hauteur est proportionnelle à son effectif. Les modalités sont placés dans l’ordre induit par la variable.

En R : Calculer le diagramme à batons d’une variable qualitative

On utilise la fonction barplot :

  • horiz spécifie si les barres sont tracées horizontalement (ce qui n’est pas le cas par défaut) ;
  • main définit le titre du graphique ;
  • xlab définit la légende en abscisse ;
  • ylab définit la légende en ordonnée ;
  • col définit les couleurs utilisées pour chaque secteur (et donc chaque modalité).
barplot(table(data$Ordinale),
        main = "Diagramme en bâtons de la variable Ordinale",
        xlab = "Modalité",
        ylab = "Effectif")

barplot(prop.table(table(data$Nominale)), horiz = TRUE,
        main = "Diagramme en bâtons de la variable Nominale",
        ylab = "Modalité",
        xlab = "Fréquence",
        col = c("blue", "red", "green"))

Remarque. On peut aussi utiliser le diagramme à batôns pour une variable quantitative discrète : 
barplot(table(data$Discrète))

6.1.2.2 Diagramme en barres

Chaque modalité est représentée dans un rectangle dont :

  • l’une des dimensions est fixe ;
  • l’autre dimension est proportionnelle à sa fréquence.

En R : Tracer le diagramme en barres d’une variable qualitative.

On utilise la fonction barplot en effectuant une conversion de la table des effectifs avec as.matrix :

barplot(as.matrix(table(data$Ordinale)),
        main = "Diagramme en barres de la variable Ordinale",
        xlab = "Modalité",
        ylab = "Effectif",
        legend.text = TRUE)

barplot(as.matrix(prop.table(table(data$Nominale))), horiz = TRUE,
        main = "Diagramme en barres de la variable Nominale",
        ylab = "Modalité",
        xlab = "Fréquence",
        col = c("blue", "red", "green"))

Remarque. Ce type de représentation graphique n’est pas adaptée à un nombre de modalités élevé : 
barplot(as.matrix(table(data$Discrète)))

6.1.2.3 Diagramme en secteurs

Chaque modalité est représentée dans un secteur de disque proportionnel à sa fréquence.

En R : Tracer le diagramme en secteurs d’une variable qualitative.

On utilise la fonction pie :

  • main définit le titre du graphique ;
  • xlab définit la légende en abscisse ;
  • ylab définit la légende en ordonnée ;
  • ‘col’ définit les couleurs utilisées pour chaque secteur (et donc chaque modalité).
pie(table(data$Ordinale),
    main = "Diagramme en secteurs de la variable Ordinale")

pie(prop.table(table(data$Nominale)),
    main = "Diagramme en secteurs de la variable Nominale",
    col = c("blue", "red", "green"))

Remarque. Ce type de représentation graphique n’est pas adaptée à un nombre de modalités élevé : 
pie(table(data$Discrète))

6.2 Indicateurs

6.2.1 Mode

Définition 6.4 Le mode de \(X\) est la modalité dont l’effectif est le plus élevé.

En R : Calculer le mode d’une variable qualitative

La fonction n’existe pas par défaut en R. Comme la fonction mode est déjà définie en R, nous allons créer la fonction modal.factor :

modal.factor <- function(x, na.rm = FALSE)
 {
   if(na.rm) { x = x[!is.na(x)] }
   levels(x)[which.max(table(x))]
 }

L’option na.rm spécifie si les valeurs manquantes doivent être retirées avant le calcul du mode (par défaut, ce n’est pas le cas).

modal.factor(data$Ordinale)
[1] "3"
modal.factor(data$Nominale)
[1] "Rouge"
modal.factor(data$Dichotomique)
[1] "Oui"

6.2.2 Moyenne des variables dichotomiques binaires

Proposition 6.1 Soit \(X\) une variable qualitative dont les seules modalités sont \(0\) et \(1\).

Alors \(\overline{X} = \dfrac{1}{N}\displaystyle\sum_{i = 1}^N x_i\) est égale à la fréquence de la modalité \(1\).
